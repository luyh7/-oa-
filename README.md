# baioo-oa-extension

本插件仅限于http://oa.info使用

- 特性
  - 在 ”我的考勤“ 右侧显示本月工作时长和本周工作时长
  - 在控制台显示每周工作时长
  - 在日历上显示每日工作时长和每周工作时长
  - 在翻页的时候更新时长显示
- Files
  - xbrowser.crx 插件压缩包，拖放到谷歌浏览器即可安装
  - xbrowser.pem 打包私钥
  - xbrowser 项目文件夹
